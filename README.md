# 🚢 Titanic Survival Prediction - Decision Tree Analysis

## 📌 Présentation du Projet
Ce projet analyse les facteurs déterminants de la survie des passagers du Titanic en utilisant des algorithmes d'apprentissage automatique (**Machine Learning**). L'objectif est de construire un modèle prédictif robuste tout en garantissant la fiabilité des données traitées.

Ce travail a été réalisé dans une optique de **qualité logicielle** et d'**optimisation des processus décisionnels**.

## 🚀 Compétences Clés Démontrées
* **Master Data Management** : Nettoyage et fiabilisation d'un jeu de données de 891+ entrées.
* **Modélisation Statistique** : Mise en œuvre d'un arbre de décision (`DecisionTreeClassifier`).
* **Analyse de Performance** : Évaluation rigoureuse via les métriques de Précision, Rappel et F1-Score.
* **Gestion de Version** : Utilisation professionnelle de Git pour le suivi du projet.

## 🛠️ Technologies Utilisées
* **Langage** : Python 3.x
* **Librairies** : 
    * `Pandas` : Manipulation et structuration des données de référence.
    * `Scikit-Learn` : Modélisation et calcul des métriques de performance.
    * `Matplotlib` / `Seaborn` : Visualisation des données et de l'arbre.

## 📊 Méthodologie & Résultats
1. **Préparation (Data Cleaning)** : Traitement des valeurs manquantes et encodage des variables stratégiques (Sexe, Classe, Tarif).
2. **Partitionnement** : Division stricte des données en **70% Entraînement** et **30% Test** pour valider la capacité de généralisation.
3. **Performance** : Le modèle atteint une **précision globale de 80%** sur des données inconnues.

### Métriques détaillées (Test Set) :
| Classe | Précision | Rappel (Recall) | F1-Score |
| :--- | :--- | :--- | :--- |
| **Global** | **0.80** | **0.78** | **0.78** |

> **Analyse Qualité** : L'écart minime entre les scores d'entraînement (82%) et de test (80%) confirme l'absence d'overfitting et la robustesse des critères de décision sélectionnés.
